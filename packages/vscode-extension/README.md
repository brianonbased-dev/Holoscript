# HoloScript VS Code Extension

Official language support for **HoloScript** (.holo, .hs, .hsplus).

## Features
- Syntax Highlighting
- Code Snippets
- AI-Powered Completions via uAA2
- Spatial Profiling

## Requirements
- uAA2 Service (optional, for AI features)

## Install
Run `code --install-extension holoscript-vscode-1.0.0.vsix`
